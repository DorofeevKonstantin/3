Реализовать любую интересную для вас игру в рамках ООП.
Позаботиться о качестве кода, об архитектуре, о расширяемости, изменяемости.

За меньшее количество баллов можно взять готовую игру на языке C++/Python и сильно модифицировать,
разобравшись в проекте, вникнув в его архтиектуру, изменив проблемные участки кода, добавить функционал.

За большее количество баллов можно реализовать игру по сети (мультиплеер).
Для простоты игроки будут находится внутри локальной сети.

За ОГРОМНОЕ количество баллов можно реализовать интерактивную игру :
- управление голосом (команды)
- управление жестами (веб-камера)
- распознавание фигур / геометрических примитивов / цветных пазлов / палочек / спичек
/ выкладывание узоров / рисование на листе бумаги и проецирование нарисованного в игре / ...
и использование распознанных объектов для выполнения игровых заданий
