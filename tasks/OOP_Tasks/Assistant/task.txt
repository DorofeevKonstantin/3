Реализовать голосового помощника.
Пользователь озвучивает простой вопрос (термин) - программа выдаёт результат поиска в гугле / википедии...
Например :
-> "питон"
<- Python — мультипарадигмальный язык программирования. Полностью поддерживаются объектно-ориентированное, структурное, обобщённое...

Задание можно самостоятельно сильно усложнить (добавить фичи: анализ нескольких источников и прочее) и получить невероятное количество баллов.
