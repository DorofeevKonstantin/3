Реализовать собственную "простую" кроссплатформенную GUI библиотеку.

- Реализовать класс Window и основные варианты взаимодействия с ним.
- Реализовать класс Button и основные события.
- ... 

Реализовать поддержку Windows и Linux систем.
Использовать нативные вызовы (Winapi, X window system), не использовать
готовые сторонние библиотеки, только системные.
https://en.wikipedia.org/wiki/X_Window_System
https://en.wikipedia.org/wiki/Xlib

Можно заработать дополнительные баллы следующим образом:
Найти одногруппника, который тоже реализовал свою библиотеку.
Провести code-review его библиотеки, отдать ему свой код на анализ.
Сам процесс code-review тоже будет оценён.
